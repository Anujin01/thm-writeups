**Date:** 5/12/2026 

**Path:** Pre-Security

## What This Room Covered

I studied the basics of the Windows and Linux CLI in this room. I learned how to navigate files and folders, locate files even without
knowing the full path, and read file contents. I also practiced finding hidden files and directories and gathered basic system information
using both the Windows and Linux command lines.

## Terms I Learned

* **CLI:** Command-Line Interface

* **CMD:** Command Prompt for Windows

| Action | Windows (CMD) | Linux |
| :--- | :--- | :--- |
| **List Files** | `dir` | `ls` |
| **Change Directory** | `cd` | `cd` |
| **Print Directory** | `cd` (no args) | `pwd` |
| **Read File** | `type` | `cat` |

## What Actually Clicked

I learned the fundamental commands to navigate between files and folders in both Windows and Linux. From my perspective,
I found the Linux CLI easier to navigate since it is widely used in cybersecurity. However, both use similar logic, which is helpful to
remember. I noticed that Windows often displays more detailed system information by default with similar commands.
