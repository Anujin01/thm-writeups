## Terms i learned

**http status code**

100-199 - information response

200-299 - success

300-399 - redirection

400-499 - client side error

500-599 - server side error (might be serious)

## What Actually Clicked

When we access a web server through the HTTP protocol, we use an HTTP request and get responses back
in a specific sequence, rather than everything at once. Using headers provides additional bits of 
information during a request, and in return, the server provides a response header. Cookies are 
small pieces of data stored on my computer; they are saved when we receive a Set-Cookie header from 
a web server.
